# Android 手势控制插件 APK

- 最低兼容：Android 8.0 (API 26)
- 开发语言：Kotlin

## 构建
- 安装 Android Studio 与 SDK、JDK 17
- 使用 Android Studio 打开项目并执行 `Build > Make Project`
- 生成的 APK 位于 `app/build/outputs/apk/`

## 权限与启用
- 悬浮窗：在应用内点击“授予悬浮窗权限”跳转系统设置授权
- 辅助功能：在应用内点击“启用辅助功能服务”进入设置开启服务
- 截屏：首次触发截屏时系统弹窗授权

## 运行
- 在配置页面为六种手势选择对应功能
- 点击“启动悬浮服务”保持后台运行并显示左右侧触控层

## 资源占用
- 服务前台运行，通知优先级低
- 不引入重型依赖，目标运行占用 < 15MB
